# Drop4Up: 屬靈生命成長 App 完整開發企劃書 (V1.3)

## 1. 核心願景與設計目標 (Vision & Goals)
Drop4Up 致力於打造一個「反多巴胺」的寧靜心靈空間。它不只是一個記錄工具，而是一個讓使用者能與內心對話、感受生命厚度的避風港。
- **設計目標**：極簡、寧靜、具反思性且情感安全。
- **情感基調**：平和、踏實、受鼓勵、無壓力（非侵略性設計）。
- **使用者狀態**：專注於當下的領受，而非社交回饋。

---

## 2. 視覺設計規範 (Design Specification)

### 🎨 色彩規範 (Color Palette)
| 項目 | 色號 (HEX) | 說明 |
| :--- | :--- | :--- |
| **背景 (Background)** | `#F2F2EE` | 中性暖灰米色。確保溫潤但不偏黃，與卡片形成低對比層次。 |
| **卡片 (Card Surface)** | `#FAFAFA` | 柔和白。與背景保持約 5% 的對比，營造細膩的隆起感。 |
| **主色 (Primary Blue)** | `#7C96A8` | 莫蘭迪藍。低飽和度藍灰色，用於主要動作按鈕。 |
| **輔色 (Accent Blue)** | `#6E8FA3` | 深色莫蘭迪藍。用於點綴或狀態切換，提升高級質感。 |

### 🧱 視覺風格與光影 (Visual Style & Lighting)
- **光源方向**：左上方 (Top-left light source)。
- **3D 質感**：進階新擬物化 (Soft Neumorphism)，強調「柔軟的表面壓印感」。
- **陰影系統 (Dual-layer Shadow)**：
    1. **外部陰影 (Outer Shadow)**：深色軟陰影，使用高模糊 (High Blur) 與低透明度。
    2. **內部高光 (Inner Highlight)**：白色的柔和內陰影，營造柔軟的凸起效果。

### 🔤 字體與排版 (Typography)
- **字體系列**：SF Pro / Inter 等潔淨的無襯線體。
- **排版邏輯**：使用較細的字重 (Light / Regular)，字體偏小 (Very small)，營造精緻、非侵略性的視覺感。
- **空間佈局**：極大比例的留白 (Negative Space)，提供視覺呼吸空間。

### 🧩 UI 組件與圖示 (UI Details)
- **圖示 (Icons)**：極簡線條感，細線條 (Thin stroke)，低對比設計，不干擾內容，維持介面空靈感。
- **按鈕 (Buttons)**：具備新擬物化凹凸質感，結合 **Haptic 觸覺回饋** 設計（當使用者按下時，陰影由「凸」轉「凹」，模擬真實物理按壓）。
- **材質質感**：模擬真實物理表面，呈現高品質產品展示質感，強調觸覺與視覺的一致性。

## 3. 資訊架構與「4 分頁 + 空間佔位」導覽設計

為了維持 4 個功能的純粹性，我們採用「虛擬 5 格」空間分配，但不增加多餘分頁。中央第 3 格設為「透明佔位」，讓凸起的 Drop 按鈕完美置中。

| 導覽格位 | 功能分頁 | 核心功能與 UI 重點 |
| :--- | :--- | :--- |
| **1 (左一)** | **🏠 首頁 (Home)** | 感性回顧：時間軸、隨機感動卡片、標籤雲。 |
| **2 (左二)** | **🧪 實驗室 (AI)** | **智慧核心**：整合搜尋標籤、編輯紀錄與 AI 高質感圖文轉化。 |
| **3 (正中)** | **💧 Drop (核心)** | **行動焦點**：明顯凸起的 3D 水滴鈕，具備凹凸觸感與 Haptic 回饋。 |
| **4 (右一)** | **[ 空間佔位 ]** | 不設分頁，維持導覽列的呼吸感與空間對稱。 |
| **5 (右二)** | **⚙️ 個人設定** | **管理中心**：成長進度、資料備份還原與系統偏好。 |

*(註：格位 4 作為空間占位或將「個人設定」稍微向右偏移，以確保中央 Drop 鈕的視覺權重。)*

## 4. 互動體驗與使用者流程 (UX Flow)

- **快速紀錄 (Quick Entry)**：短按 Drop 鈕彈出文字輸入；長按則啟動語音辨識紀錄。
- **微互動 (Micro-interactions)**：紀錄存檔後，內容化作小水滴落入時間軸；隨機回顧時，卡片由遠而近漸顯。
- **寧靜轉場 (Transitions)**：切換頁面採用「淡入 (Fade) + 輕微縮放 (Scale)」，避免視覺跳躍感。

---

## 5. AI 視覺轉化邏輯 (Soul Lab)

- **非生成式文字**：絕對保留使用者原始輸入，AI 嚴禁改寫或生成文字。
- **高質感圖文產出**：
    - **AI 任務**：根據紀錄情緒生成抽象、莫蘭迪色調的背景圖（如雲霧、紋理、光影）。
    - **多層次排版**：前端 UI 將文字以多層次、帶光影的字體疊加於背景圖，並配合磨砂玻璃遮罩。

---

## 6. 渲染規格與展示 (Rendering & Presentation)
為了確保設計方案在開發前能完美呈現其高品質質感，Mockup 需符合以下規格：
- **解析度**：8K 超高解析度，展現細膩的光影細節。
- **呈現方式**：光影寫實的 UI Mockup，以前視角 (Front View) 展示於 iPhone 風格的裝置中。
- **質感要求**：高端產品簡報等級，需包含細膩的 **環境遮蔽 (Ambient Occlusion)** 與真實的柔和陰影。
