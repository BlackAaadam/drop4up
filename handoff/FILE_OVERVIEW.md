# File Overview

## Root files

- `README.md`: how to use this package.
- `AGENTS.md`: repository-level Codex instructions. Put this at repo root.
- `CODEX_START_PROMPT.md`: paste this to Codex first.
- `TASKS.md`: milestone implementation plan.

## Docs

- `docs/00_source_of_truth.md`: V0.3 product and design source of truth.
- `docs/01_codex_restart_strategy.md`: how to restart implementation cleanly in Codex.
- `docs/02_flutter_inset_box_shadow_guide.md`: exact dependency/import/state guidance.
- `docs/03_soft_surface_rendering_rules.md`: visual rendering rules and tuning order.
- `docs/04_screen_implementation_plan.md`: screen requirements.
- `docs/05_acceptance_checklist.md`: visual/function acceptance checklist.
- `docs/06_branch_and_worktree_guide.md`: branch and commit strategy.
- `docs/07_common_failure_modes.md`: common issues and fixes.

## Flutter snippets

- `flutter_snippets/pubspec_additions.yaml`: dependency snippet.
